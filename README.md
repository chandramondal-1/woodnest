# WOODNEST Premium Storefront

This workspace now includes a static premium storefront built with plain `HTML`, `CSS`, and `JavaScript`.

## Main entry

- Open `index.html` directly in the browser for the static website
- Product images are stored in `assets/products`
- Styling lives in `styles.css`
- Catalog data and filtering logic live in `script.js`

## Included

- Premium hero section with luxury visual treatment
- Category-based browsing for `L Shape`, `Sofa Sets`, `1 Seater`, `2 Seater`, `3 Seater`, `Sofa Cum Bed`, `Luxury Lounge`, and `Accent Chairs`
- Real product catalog populated from the provided WOODNEST screenshots
- Search and category filters
- Product detail modal for each item

## Note

Older Next.js starter files are still present in the workspace, but the requested static website version starts from `index.html`.
